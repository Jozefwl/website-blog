## Building

```
docker-buildx build \
  --platform=linux/amd64 \
  -t harbor.waldoserver.top/cloud/website-blog:0.2.2 \
  --no-cache \
  .
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
